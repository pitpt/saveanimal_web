
タイトル：


概要：
全国の保健所に引き取られている動物を統括管理するシステム


使用言語：
HTML
CSS
JS
PHP

データベース：
MySQL


基本システム：

サイトユーザー（一般人）は動物を条件検索できる。（Google Map APIを使った地図ベースのUIも作成？）
クライアント（保健所）は　専用フォームから動物の情報を登録、管理できる。

サーバーから持ってくるデータはjsonデータとしソート条件によってPHPでMySQLにアクセスにJSONパースされたテキストを返す。
masterブランチでは例を置いて開発を進める。

運用ブランチ：
master:
サーバーサイドを抜きにしたhtml、CSS、JS
CSSにてデザインの充実等を行う。
masterブランチではjsonファイルの例を置いて開発を進める。

server:
サーバーサイド、PHP,MySQLの開発を行う。
ローカル環境ではそれぞれxampp等をインストールし開発を進める。
